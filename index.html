About

This project is a work in progress by Maxwell Bourcier as a Capstone Project for Genspark. This project utilizes the knowledge gained from the Cohort in its entirety. This opprotunity has lead to growing as a programmer and becoming confident with coding and its many uses / paths.

This project ultimately began as a fix for a personal issue while recording workouts out at the gym.

FAQ

Q: Why?

A: I always have my phone, but never have a notebook to record my workouts. This is my solution for that, while not having to use a boring notepad on the phone itself. Of course this exists already, but not to this extent.

Q: What Does It Do?

A: The project's function is to let anyone quickly reference past workouts, store workout schedules into a database from an image, input your workout ones as an image (of a notebook or a screenshot of a workout in notes), compare numbers with previous weeks and see a trend develop as data is inserted. And ultimately give the user a site to go back to to collect, manipulate, view or whatever the user wishes. A window into the Database.

Q: Database?

A: Yes. Using SQL through Microsoft SQL Management Studio, all information will be stored and accessed through there. This immediately allows for CRUD operations to be performed which is invaluable for data storage and manipulation with ease.

Q: Will I Be Able to Use This Too?

A: This project will begin as a singular user website, protected by password. But will allow for multiple users within the year!
